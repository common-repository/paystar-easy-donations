== Description ==

with this plugin, you can easily add paystar.ir donation form to your pages/posts
با این پلاگین میتونید فرم های پرداخت / اهدا وجه رو به همراه درگاه پرداخت پی استار به سایتتون اضافه کنید


== Installation ==
 
just enter paystar.ir terminal (API-Key)
راه اندازی و پیکربندی افزونه نیاز به تنظیمات خاصی نیست. فقط کافیه کد ترمینال پذیرندگی خودتون رو در تنظیمات افزونه وارد کنید
برای ورود به صفحه تنظیمات افزونه از منوی حمایت مالی آسان ، زیرمنوی تنظیمات رو کلیک کنید
